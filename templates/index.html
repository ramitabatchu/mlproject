Student Math Exam Performance Predictor

About This Project

This web application predicts a student's math exam performance based on various factors such as gender, ethnicity, parental education, lunch type, and test preparation. The prediction is powered by a machine learning model trained on historical student data.

Machine Learning Model

The model used in this project is a Gradient Boosting Regressor, an ensemble learning method that builds models sequentially and improves prediction accuracy by correcting errors from previous iterations. Other models like Random Forest, Decision Trees, and XGBoost were also evaluated, but Gradient Boosting performed best.
